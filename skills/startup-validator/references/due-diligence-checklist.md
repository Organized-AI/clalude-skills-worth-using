# Comprehensive Due Diligence Checklist

## Financial Due Diligence

### Revenue Verification
- [ ] 3 years of audited financial statements
- [ ] Revenue recognition policies documented
- [ ] Customer contracts reviewed (top 20)
- [ ] Revenue concentration analysis
- [ ] Deferred revenue schedule
- [ ] Accounts receivable aging
- [ ] Bad debt reserves
- [ ] Sales pipeline validation

### Cost Structure
- [ ] COGS breakdown by category
- [ ] Operating expense trends
- [ ] Employee compensation structure
- [ ] Marketing spend efficiency
- [ ] R&D investment breakdown
- [ ] Capital expenditure history and plans
- [ ] Working capital requirements

### Profitability & Cash Flow
- [ ] Gross margin trends
- [ ] Operating margin analysis
- [ ] EBITDA calculation and trends
- [ ] Free cash flow generation
- [ ] Cash conversion cycle
- [ ] Burn rate and runway
- [ ] Path to profitability

### Financial Projections
- [ ] Financial model review
- [ ] Key assumption validation
- [ ] Sensitivity analysis
- [ ] Scenario planning (base, upside, downside)
- [ ] Historical accuracy of projections
