# Professional Pitch Deck Design Guidelines

## Core Principles

### 1. Clarity Over Cleverness
- Clear message beats clever design
- One main point per slide
- Visual hierarchy guides the eye
- White space is your friend

### 2. Data Visualization
- Choose right chart for data
- Label axes clearly
- Use color purposefully
- Keep it simple

### 3. Consistency
- Same fonts throughout
- Consistent color palette
- Uniform spacing
- Repeated design elements

## Typography

### Font Pairing

**Option 1: Professional**
- Headings: Montserrat Bold
- Body: Open Sans Regular
- Data: Roboto Mono

**Option 2: Modern**
- Headings: Poppins SemiBold  
- Body: Inter Regular
- Data: JetBrains Mono

**Option 3: Classic**
- Headings: Merriweather Bold
- Body: Lato Regular
- Data: Source Code Pro

### Font Sizes
- **Slide Title:** 32-36pt
- **Section Header:** 24-28pt
- **Body Text:** 18-20pt
- **Caption/Data:** 14-16pt
- **Footer:** 10-12pt

### Font Rules
- Maximum 2 font families per deck
- Use bold for emphasis, not underline
- Avoid all caps except for headings
- Maintain consistent line spacing (1.5x)

- **Primary:** Dark Blue (#1E40AF)
- **Secondary:** Teal (#14B8A6)
- **Accent:** Orange (#F97316)

### Color Usage Rules
- Use primary color for main elements

- Accent for calls-to-action

## Layout

### Slide Dimensions
- **Standard:** 16:9 (1920x1080px)
- **Alternative:** 4:3 (1024x768px) for older projectors

### Margins & Spacing
- **Margins:** 1" (72px) all sides
- **Header:** 0.5" (36px) from top
- **Footer:** 0.5" (36px) from bottom
- **Content padding:** 0.75" (54px)

### Grid System
- Use 12-column grid for alignment
- Align elements to grid
- Consistent spacing between elements
- Visual rhythm throughout deck

## Chart Types & When to Use

### Line Charts
**Use for:** Trends over time
- Revenue growth
- Customer acquisition
- Metric trends

**Best Practices:**
- Label axes clearly
- Limit to 3-4 lines maximum
- Use distinct colors
- Add data labels for key points

### Bar Charts
**Use for:** Comparisons
- Revenue by quarter
- Market share
- Product performance

**Best Practices:**
- Horizontal bars for long labels
- Sort by value (high to low)
- Use same color for same category
- Add value labels on bars

### Pie Charts
**Use for:** Part-to-whole relationships
- Market share
- Revenue mix
- Customer segments

**Best Practices:**
- Limit to 5-6 segments max
- Start at 12 o'clock
- Order by size
- Label clearly or use legend

### Waterfall Charts
**Use for:** Sequential changes
- Valuation buildup
- Budget breakdown
- Synergy analysis

**Best Practices:**
- Color code increases/decreases
- Label each step
- Show net change clearly

### Scatter Plots
**Use for:** Correlations
- Price vs. volume
- Risk vs. return
- Competitive positioning

**Best Practices:**
- Label quadrants
- Highlight your position
- Use size for third dimension

## Best Practices

### Content
- **6x6 Rule:** Max 6 bullets, 6 words each
- **One Message:** One key takeaway per slide
- **Tell Story:** Narrative flow between slides
- **Show, Don't Tell:** Use visuals over text

### Design
- **Consistency:** Same style throughout
- **White Space:** Don't overcrowd slides
- **Hierarchy:** Most important info largest
- **Alignment:** Everything aligned to grid

### Data
- **Simplify:** Remove unnecessary detail
- **Highlight:** Draw attention to key data
- **Context:** Provide comparison points
- **Source:** Cite data sources

### Images
- **High Quality:** 300 DPI minimum
- **Relevant:** Support your message
- **Consistent Style:** Same photo treatment
- **Attribution:** Credit photographers

## Common Mistakes to Avoid

### ❌ Too Much Text
- Slides should support, not repeat your speech
- Use bullet points, not paragraphs
- If it's more than 30 words, it's too much

### ❌ Poor Color Contrast
- Light text on light background
- Red/green for colorblind accessibility
- Busy backgrounds

### ❌ Inconsistent Design
- Different fonts on different slides
- Varying spacing and alignment
- Mixed chart styles

### ❌ Chart Junk
- Unnecessary 3D effects
- Too many gridlines
- Decorative elements that don't add value

### ❌ Unclear Hierarchy
- All text same size
- No visual emphasis
- Hard to scan quickly

## Deck-Specific Guidelines

### M&A Acquisition Deck
- **Tone:** Professional, data-driven
- **Colors:** Conservative (navy, green, gold)
- **Charts:** Financial (waterfall, line, bar)
- **Length:** 12-15 slides max

### Startup Pitch Deck
- **Tone:** Energetic, visionary
- **Colors:** Bold (blue, purple, pink)
- **Charts:** Growth-focused (line, bar)
- **Length:** 10-12 slides max

### Executive Summary
- **Tone:** Concise, high-level
- **Colors:** Neutral (blue, gray)
- **Charts:** Key metrics only
- **Length:** 5-7 slides max
