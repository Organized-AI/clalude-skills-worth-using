# M&A Due Diligence Checklist

## Financial Due Diligence

### Revenue Validation
- [ ] 3 years of financial statements (audited if available)
- [ ] Revenue recognition policies documented
- [ ] Customer contracts reviewed
- [ ] Deferred revenue schedule
- [ ] Revenue by customer/product/geography
- [ ] Refund and chargeback history
- [ ] Backlog and pipeline analysis

### Profitability Analysis
- [ ] Gross margin by product line
- [ ] COGS breakdown and trends
- [ ] Operating expense analysis
- [ ] EBITDA calculation and adjustments
- [ ] One-time expenses identified
- [ ] Working capital requirements
- [ ] Burn rate and runway calculation

### Balance Sheet Review
- [ ] Asset verification
- [ ] Liability schedule
- [ ] Debt covenants review
- [ ] Off-balance sheet items
- [ ] Related party transactions
- [ ] Contingent liabilities

### Cash Flow Analysis
- [ ] Operating cash flow trends
- [ ] Free cash flow calculation
- [ ] CapEx requirements
- [ ] Seasonality patterns
- [ ] Cash collection metrics
- [ ] Payment terms analysis

## Operational Due Diligence

### Customer Analysis
- [ ] Top 20 customers (revenue concentration)
- [ ] Customer retention/churn analysis
- [ ] Customer satisfaction scores
- [ ] Contract terms and renewal rates
- [ ] Customer reference calls
- [ ] Pipeline and bookings forecast

### Product & Technology
- [ ] Product roadmap review
- [ ] Technology stack documentation
- [ ] Technical debt assessment
- [ ] Code quality review
- [ ] Security audit
- [ ] Scalability analysis
- [ ] IP and patent portfolio

### Go-to-Market
- [ ] Sales process documentation
- [ ] Marketing effectiveness
- [ ] CAC and LTV calculations
- [ ] Sales team productivity
- [ ] Channel partner agreements
- [ ] Competitive positioning

### Organization
- [ ] Org chart and roles
- [ ] Key employee retention risk
- [ ] Compensation benchmarking
- [ ] Culture assessment
- [ ] HR policies and compliance
- [ ] Employee handbook review

## Legal Due Diligence

### Corporate Structure
- [ ] Articles of incorporation
- [ ] Cap table and ownership
- [ ] Board minutes review
- [ ] Shareholder agreements
- [ ] Voting rights analysis
- [ ] Option pool and vesting

### Contracts
- [ ] Material customer contracts
- [ ] Vendor and supplier agreements
- [ ] Partnership agreements
- [ ] Lease agreements
- [ ] Employment agreements
- [ ] Non-compete and confidentiality

### Compliance
- [ ] Regulatory compliance status
- [ ] Industry licenses and permits
- [ ] Data privacy compliance (GDPR, CCPA)
- [ ] Export control compliance
- [ ] Environmental compliance
- [ ] Litigation history

### Intellectual Property
- [ ] Trademark registrations
- [ ] Patent portfolio
- [ ] Copyright ownership
- [ ] Trade secrets protection
- [ ] IP assignment agreements
- [ ] Open source license compliance

## Strategic Fit Assessment

### Synergy Analysis
- [ ] Revenue synergy opportunities
- [ ] Cost synergy opportunities
- [ ] Cross-sell potential
- [ ] Upsell opportunities
- [ ] Geographic expansion
- [ ] Technology leverage

### Integration Planning
- [ ] Day 1 readiness checklist
- [ ] 100-day integration plan
- [ ] Systems integration requirements
- [ ] Organization design
- [ ] Communication plan
- [ ] Culture integration approach

### Risk Assessment
- [ ] Key person dependencies
- [ ] Customer concentration risk
- [ ] Technology obsolescence risk
- [ ] Competitive threats
- [ ] Regulatory changes
- [ ] Integration complexity
